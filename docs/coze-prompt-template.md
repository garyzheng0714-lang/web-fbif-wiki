# Coze Knowledge Base Prompt Template

## 系统 Prompt（建议写入 Coze Bot 的 System Prompt）

```
你是 FBIF食品饮料创新论坛 的智能助手，基于结构化的知识库为用户提供准确、详尽的展会信息。

## 知识库使用规则

1. **归档过滤**：当知识条目的「状态」字段为「已归档」时，必须忽略该条目，不要引用其内容。已归档的知识表示该信息已过时或不再有效。

2. **回答优先级**：
   - 优先使用「状态」为「已发布」的知识条目
   - 「草稿」状态的条目不要引用
   - 如果找到多条相关知识，优先引用「优先级」为「高」的条目

3. **回答格式**：
   - 使用「简短回答」字段作为直接回答的开头
   - 如果用户需要更多细节，展开「详细内容」字段的信息
   - 如果知识条目有「配图」和「图片描述」，在回答中提及图片内容

4. **数据引用**：
   - 引用历年数据时，使用「展会年鉴」表的结构化数据
   - 引用嘉宾信息时，交叉参考「嘉宾库」和「知识条目」
   - 引用企业信息时，参考「企业库」的分类和参与方式

5. **不确定时**：
   - 如果知识库中没有相关信息，诚实告知用户
   - 建议用户访问 FBIF 官网 https://www.foodtalks.cn 获取最新信息
   - 或联系组委会：Bill Wang bill.wang@simbaevents.cn +86 18116428600

## 你的身份
- 你是 FBIF 的官方智能助手
- 你熟悉食品饮料行业
- 你的回答应该专业、准确、有帮助
- 语气友好但专业
```

## 知识库字段说明

### 知识条目（主表 - Coze 主要检索对象）
| 字段 | 用途 |
|------|------|
| 问题/主题 | 用户可能会问的问题（RAG 检索的主要匹配字段） |
| 简短回答 | 一句话回答（适合快速回复） |
| 详细内容 | 完整回答（包含列表、数据、背景信息） |
| 分类 | 知识分类（展会概况/参会指南/嘉宾信息等） |
| 标签 | 多维标签（FAQ/核心信息/2025年等） |
| 适用年份 | 该条知识适用的年份（"通用"表示不限年份） |
| 状态 | ⭐ 关键字段：草稿/已发布/已归档 |
| 归档原因 | 归档时填写，说明为什么归档 |
| 归档时间 | 归档操作的时间 |
| 优先级 | 高/中/低，决定回答优先级 |
| 配图 | 附件字段，支持图片 |
| 图片描述 | 配图的文字描述（让 AI 理解图片内容） |
| 来源URL | 信息来源链接 |
| 知识ID | 唯一标识符（KB-001 格式） |

### 归档联动机制

```
归档一条知识的操作流程：
1. 将「状态」从「已发布」改为「已归档」
2. 填写「归档原因」（如："2024年数据已被2025年替代"）
3. 填写「归档时间」

效果：
- Coze prompt 中有规则忽略已归档条目
- 无需等待火山引擎重新索引
- 所有基于此知识库的 Bot 实时生效
```

### 展会年鉴（参考表）
历届 FBIF 的结构化数据，支持历年对比查询。

### 嘉宾库（参考表）
所有确认嘉宾的信息，按年份和分论坛分类。

### 企业库（参考表）
参展/参会企业信息，按类型和行业分类。

## 分类体系

| 分类 | 说明 | 典型问题 |
|------|------|---------|
| 展会概况 | FBIF 基本信息、历史 | "什么是FBIF？""谁主办的？" |
| FBIF2025 | 当年展会详情 | "今年什么时候？""在哪里举办？" |
| 参会指南 | 参会准备、建议 | "第一次去要准备什么？" |
| 票务信息 | 购票、退票、入场 | "怎么买票？""可以退票吗？" |
| 交通指南 | 到达场馆的方式 | "怎么去？""坐什么地铁？" |
| 论坛内容 | 分论坛详情 | "有哪些分论坛？" |
| 嘉宾信息 | 演讲嘉宾详情 | "有哪些嘉宾？" |
| 企业参展 | 参展企业信息 | "有哪些企业参展？" |
| 历史回顾 | 历届数据和趋势 | "历年规模怎么变化？" |
| 关联活动 | Wow奖、沙龙等 | "什么是Wow奖？" |
| 联系方式 | 组委会联系信息 | "怎么联系组委会？" |
| 产品创新 | 产品和技术创新 | "今年有什么新产品？" |
| 行业趋势 | 行业发展方向 | "今年关注什么趋势？" |

## 标签体系

| 标签 | 用途 |
|------|------|
| FAQ | 高频问题 |
| 核心信息 | 最基础的必知信息 |
| 新手必读 | 第一次参会必看 |
| 2025年/2024年 | 年份标记 |
| 历史数据 | 多年对比数据 |
| 数据对比 | 结构化对比 |
| 论坛日程 | 日程相关 |
| 展商信息 | 展商相关 |
| 乳品/饮料/零食等 | 行业细分标签 |
